Setup Cardano Protocol Handler

This extension can handle web+cardano: URLs to make Cardano transactions easier and more secure.

üîê Permission Required

To handle web+cardano URLs, this extension needs to be registered as the default protocol handler for web+cardano links.

Example URL: web+cardano://send?amount=10&address=addr1...

What this means:

  • When you click a web+cardano link, this extension will handle it
  • You'll see a secure interface to approve or deny transactions
  • Your wallet information stays safe and private
  • You can revoke this permission at any time
